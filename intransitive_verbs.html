Intransitive Verbs

Intransitive verbs are pretty easy to learn. These types of verbs only require a subject. Regardless of the type of nouns, any sentence can be made up of only a verb. This is because each conjugation communicates a single possible subject (and object in the context of transitive verbs). The conjugation table for intransitive verbs is as follows:

pronoun prefix suffix example translation
Niin n= - ninibaa I sleep
Niinawind n= =min ninibaamin We (exclusive) sleep
Giinawind g= =min ginibaamin We (inclusive) sleep
Giin g= - ginibaa You sleep
Giinawaa g= =m ginibaam Y'all sleep
Wiin - - nibaa He/She/It sleeps
Wiinawaa - =wag nibaawag They sleep
